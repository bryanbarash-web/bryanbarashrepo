# Dutchie Brand Guidelines — Hard Rules for Web Content

**MANDATORY: All web content created in this repo MUST follow the Dutchie Brand Guidelines (see `decks/Brand_Guidelines_2026.pptx`).** No exceptions.

## Core Colors

| Color        | Hex       | RGB            | Usage                     |
|-------------|-----------|----------------|---------------------------|
| Dark Green  | `#042017` | 4/32/23        | Primary (backgrounds, text) |
| Sand        | `#F1EBD6` | 240/232/214    | Primary (backgrounds, text) |
| Medium Green| `#003D29` | 0/61/41        | Secondary                 |
| Purple      | `#5E324E` | 94/50/78       | Secondary                 |
| Light Green | `#6AB448` | 106/186/72     | Accent only (minimal use) |
| Yellow      | `#FFC02A` | 255/192/42     | Accent only (minimal use) |

## Preferred Color Combinations

- **Dark Green background + Sand text** (primary)
- **Sand background + Dark Green text** (primary)
- Sand on Medium Green, Sand on Purple, Dark Green on Light Green, Dark Green on Yellow (accessible alternates)

## Typography

- **Primary font:** Matter
- **Web fallback:** Inter (Google Font)
- **System fallback:** Arial, then any web-safe sans-serif
- Weights: Regular, Medium, Semi Bold, Bold

## Logo Rules

- Use the official logotype (icon + "dutchie" wordmark) as the primary logo
- Light logo on dark backgrounds, dark logo on light backgrounds
- Maintain clear space of at least 1/2 the logomark height around the logo
- NEVER recreate the logo — always use provided source files
- The logomark is called the "Group Hug"

## Slide Deck Template

**MANDATORY: All PowerPoint decks generated by Tilda MUST use `decks/dutchie deck template.pptx` as the template.**

When calling Tilda's `pptx_generate_pptx` tool, always pass:
- `template_path`: path to `decks/dutchie deck template.pptx` in this repo
- This ensures all generated decks inherit the correct master slides, layouts, theme colors, and fonts

## Logo Assets

- `images/logo-light.png` — Sand/cream logotype for dark backgrounds
- `images/logo-dark.png` — Dark green logotype for light backgrounds
- Always use the appropriate version based on background color

## Enforcement

When building any HTML, CSS, or web page in this repo:
1. Use the brand color palette above — no arbitrary colors
2. Use Inter (or Matter if available) as the font family
3. Preferred layout: Dark Green backgrounds with Sand text, or Sand backgrounds with Dark Green text
4. Include the Dutchie logo where appropriate
5. Ensure accessible color contrast per the approved pairings

When generating any slide deck:
1. Always use `decks/dutchie deck template.pptx` as the template
2. Follow the brand color palette for any custom elements
3. Use the official logo assets from this repo
